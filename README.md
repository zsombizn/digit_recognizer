# digit_recognizer
Digit recogniser based on [MNIST](http://yann.lecun.com/exdb/mnist/) data in C.

### Installation:

Clone the repository:

```bash
git clone https://github.com/zsombizn/digit_recognizer.git
```

### Compilation:

Use make:

```bash
make
```
